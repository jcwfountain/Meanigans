Page 4 - Generation

Most bombs are classified into four different styles, called their generation.

Bomb generations can be identified with the last two letters in the serial code, typically in the corner of a device.

For the following information, please note that letters A-M are considered the first half of the alphabet.

First generation bombs use two of the same letter

Second generation bombs have two letters, both of which are in the first half of the alphabet.

Page 6 - Security Code

This guide will make frequent references to a security code.

The security code can easily be found by finding the sum of the digits in the serial number
